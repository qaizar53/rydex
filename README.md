# 🚖 Rydex - Full Stack Ride Booking Platform

Rydex is a modern full-stack ride booking platform built with **Next.js**, **TypeScript**, and **MongoDB**. The application provides a seamless experience for passengers, drivers, and administrators with real-time location services, ride management, secure authentication, and responsive dashboards.

---

## 🌟 Features

### 👤 User
- Register and Login
- Book rides
- Select pickup and destination locations
- Live route visualization
- Fare estimation
- View booking history
- Real-time ride status
- Responsive UI

### 🚗 Driver / Partner
- Partner registration
- Vehicle onboarding
- Upload verification documents
- Manage ride requests
- View earnings
- Driver dashboard

### 🛠 Admin
- Admin Dashboard
- Manage users
- Manage drivers
- Monitor bookings
- Earnings analytics
- Platform management

---

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/rydex.git
```

Move into the project

```bash
cd rydex
```

Install dependencies

```bash
npm install
```

Create a `.env.local` file and add your environment variables.

Example:

```env
MONGODB_URI=
JWT_SECRET=
NEXT_PUBLIC_GEOAPIFY_API_KEY=
NEXT_PUBLIC_BASE_URL=
```

Run the development server

```bash
npm run dev
```

Visit

```
http://localhost:3000
```
